# CAPS AI Frontend 2.0

Modernized frontend shell for CAPS AI.

## Prerequisites
- Node.js 20+

## Setup
1. `npm install`
2. Create `.env` from `.env.example`
3. Start dev server: `npm run dev`

## Environment
- `VITE_API_BASE_URL` (default: `http://localhost:8000/api/v1`)

## Build
- `npm run build`
- `npm run preview`
